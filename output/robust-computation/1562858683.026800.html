# Alan Zaffetti 2019-07-12 18:20
I and others are actively thinking about this and treating it like an interaction design problem. As a user obviously I want interactive commands to feel snappy even if the backing process is a churning, long running ecosystem that spreads the area the size of Kansas.

In our work with GPU machines we are using compute shaders to implement both the former and latter process. This can be run on a subset of local sites (processing) and displayed back to the user as a pictuire or in aggregate. One concrete example would be counting the number of data cells that match a certain pattern for instance.
