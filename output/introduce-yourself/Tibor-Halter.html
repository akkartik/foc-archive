# Tibor Halter 2019-10-25 06:06
Hi all! I've been working on a web development platform called Boomla for 14 years, I'm sort of coming out of the bush these days. https://boomla.com/ I also call it a Website OS as it is a platform where websites are first-class citizens.
It is a server-side platform, with its own object-oriented filesystem, functional execution model, package management solution, etc. It can currently be programmed in JavaScript on the server, but it will support multiple languages in the future.
It also provides a drag&drop user-interface for end-users to let them build websites. But that's optional, you could build a different frontend for it.
It has a web based IDE and lots of documentation. I made the decision early to focus on the platform and not write any new programming language. :smile:
It was designed to be low on maintenance. The platform is fast and rock solid stable. Developers tend to agree that the most interesting part of the system is the object oriented filesystem with lots of interesting properties.
I'm glad for the existence of the group, looking forward to interesting conversations!
