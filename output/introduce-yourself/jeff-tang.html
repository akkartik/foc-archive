# jeff tang 2020-04-28 09:36
Helloo, I’m Jeff. Been lurking here for a month or so but after reading the community survey transcript I realized I should speak up. You see, my current FoC project is building an open-source version of Roam Research, but I’m mainly just doing this to learn Clojure. Why? Because what I really want to make is a chat client built on top of a graph database that supports bi-directional links and multiple indexes (i.e. chronologically of course but also by tags that can organically emerge as channels similar to Zulip).

Conversation is personally my favorite Tool for Thought, but we haven’t quite cracked it digitally. Can we have frictionless divergent streams, but also refined, retrievable evergreen posts? This message by @Mariano Guerra sums it up so well, and I have to agree… I’d be so sad if I never found it! Looking forward to continuing the conversation :wink: here and on Twitter if you’re into that sort of thing
