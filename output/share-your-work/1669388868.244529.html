I saw the ‘AI found a bug in my code’ on HN and I wanted to try this out on my code. Since the tool was not open sourced, I decided to build it from scratch myself :male-technologist::sweat_smile:.

So I present to you Suspicious (https://github.com/sturdy-dev/suspicious)

It’s a fully local CLI app which analyses your source code file. It then shows you specific parts of the code where it would have done something else.
Demos & screenshots in the GitHub readme. Let me know if it catches anything good for you :slightly_smiling_face:
