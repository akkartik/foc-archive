# Paul Shen 2020-10-10 10:02
Hi! My name's Paul and I love the prompt "future of coding". In the past, I've worked at Facebook and led product infrastructure at Discord. I actually left my job to explore ideas in programming/tools/education and learn more about PL and other things.

Being interested in code/design led me towards frontend/product engineering. Having spent some time here, I feel the tools and levels of abstractions are great but have so much more potential. I'm excited for a day styling (css, etc) is mostly handled in the design tool. Working with code as text files seems primitive. I have many more thoughts and questions :stuck_out_tongue:

Some links: Investigating React memory leak at Discord, Monad tutorial for JS devs, Visualizing array.splice, twitter @_paulshen
Excited to be here!
