Forgot to post this, what with all the christmas chaos…

In just 16 hours (at time of writing, of course, you lovely pedants) we'll have the final FoC virtual meetup of 2025.

Here's the Luma page if you want to RSVP for a reminder. The meetup starts at 18:00 UTC.

We'll have 3 demos:

• I'm going to demo a little tangible sandbox I made for playing with 1-bit error-diffusion dither kernels, since these things are inherently spatial in surprising & fun ways
• @Jasmine Otto is going to show Dendryscope, a tool "meant for narrative designers to playtest their games with an intelligent interface, [which] happens to be a fun use case for answer set programming via clingo in the browser."
• @Beni Cherniavsky-Paskin will "demo (and try to motivate) some self-modifying code ideas I've been playing with"
Would love to see you there, especially if anyone happens to be tuning in at midnight local time :)

When the time comes, join here: https://ivanish.ca/zoom
