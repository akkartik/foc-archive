# Konrad Hinsen 2019-10-06 02:23
One of the things we should acknowledge sacrificing (because everyone has been doing it tacitly for a long time) is general-purpose programming. That's a useful theoretical concept (Turing machines etc.), but every real programming system (hardware plus development tools) makes choices that render it impractical for many applications. And once you give up the general-purpose dream, maybe Turing completeness becomes less relevant as well, given that it creates so many opportunities for nasty bugs (see https://hal.archives-ouvertes.fr/hal-02071770).
