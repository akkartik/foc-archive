I'm not sure what games you were playing, but there are many examples of games where you don't have to go back to 'square 1' to edit game behavior. It's true there aren't many games where creation and editing is a first class feature, but in most video games (even in games were simulation and immersion are important) its not the goal, and encouraging free modification would "break" the game. You can make arguments that for single player games that doesn't matter, but it does end up removing challenge.

That said a lot of games support modding as an important thing, even if you have to download a mod sdk, and editing isn't completely intuitive, etc. This includes major, super popular games that begat even more popular games through modification (Warcraft 3 -> Dota, Half-Life -> Counterstrike, ARMA 3 -> PUBG). Media Molecule's games Little Big Planet and the upcoming Dreams make creation first class, so you probably want to check those out if that's your main frustration. This is a bad example coming from someone that's not really familiar with video game history or design. It makes sense coming from someone that remembers some random game that they either couldn't edit or didn't know how, and that's a valid complaint, but its not representative of games as a whole.

# Scott Anderson 2018-10-08 15:30
Halo is a great example of a game series that has supported modification in a first class way, even on console which is rare due to technical and market constraints. I don't think the original xbox version of Halo had modification, but Halo 3 allowed for custom game modes and had forge (https://www.halopedia.org/Forge) which allowed you to customize maps. With Halo its tricky though because eventually you run into the technical constraints of console games, unless you build your game from the ground up to also be an editor and programming environment (at the sacrifice of quality), console games run into memory, compute and security issues. Console games are made on PCs that are significantly more powerful than the consoles they run on, and often have build processes that do hours or days of precomputation using distributed build processes (for example ray tracing for lighting). Once again you can build, but the goal of most console games (and the desire of the market) is not creative expression.
