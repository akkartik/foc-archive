# Robert Butler 2020-11-17 18:58
My quest to build a truly personal computer has hit a major milestone. I finally have a working reference hardware implementation for my uCISC processor at the TinyFPGA Bx board. It runs a battery of instruction tests and computes fib(24) successfully at 16MHz (4 clocks per instruction). See the readme and reference folder here: https://github.com/grokthis/ucisc/tree/master/hardware
