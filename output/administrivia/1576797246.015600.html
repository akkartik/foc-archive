# Shalabh 2019-12-19 15:14
Anybody else here use hypothes.is? I'm wondering if that's an alternative 'shadow' community that we can try. The basic idea is it overlays on top of any webpage or PDF and lets you annotate parts of it with text and tags. You can reply to annotations as well. Here's an example of how it looks: https://hyp.is/uIQ3RiK0Eeq_kBNudD0B8Q/futureofcoding.org/. It uses PDF fingerprints so the location of the PDF doesn't matter (could even be loaded from a local file) - the annotations will show up.

The advantage is it is kind of lightweight to adopt - doesn't need us to set up a server. Doesn't even need an extension in Firefox, you can just install a bookmarklet. To try it out we could create a FoC group on hypothesis and paste the invite link here. You also get a feed view of everything that's being annotated, e.g. https://hypothes.is/users/shalabh.
