# Mike Innes 2026-02-27 06:37
Hi all! I've been lurking for a while (and have met some of you on donut calls). My name's Mike, and I make programming languages and tools. I got started early on with Julia and built a bunch of tools for interactive development and machine learning. I'm proudest of Flux, which influenced TensorFlow 2 and has been used for lots of cool research. Nowadays I'm working on a little project called Raven, which I'm pitching as a better way to use WebAssembly and as an escape hatch from JS. I'm excited for what we can do by making powerful interactive languages accessible everywhere (explorable explanations? figma for code?), and resonate with ideas like malleable computing. It's early days but I think there's a lot of potential in those areas.

I'll keep it short, but I'm more than happy to talk more, and to hear about any overlapping work :D I'd like to participate in more of the meetups in future too (sadly didn't see the last one in time). Glad this place exists!
