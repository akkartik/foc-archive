# Doug Moen 2019-12-06 10:19
I'm not sure what to make of "no operating system" claims. Back in the day there was a more liberal definition of "operating system". At one point, a working informal definition of OS was whatever software was loaded into memory at boot. Nowadays, it seems that "operating system" is synonymous with "Unix", so if an OS doesn't have the same architecture and layering as Unix, then it isn't an OS. I could be mischevious, and claim that Unix is an image based software environment written from the ground up in C. There's no OS, because it is C all the way down, right?
