# Ivan Reese 2018-09-28 10:28
My first reaction to this was.. "I'd rather just drag vertices around, thank you very much."

Then after giving it a moment to sink in, I realized.. this is actually quite powerful. In fact, it perfectly matches the abstraction behind SVG paths, so you could create this UI atop a traditional points-and-edges / bezier based vector graphics system as an alternative mode of interacting with the graphic. This approach is more structured, whereas the points-and-edges approach is more expressive. This might be a neat way to tie a toolset like Sketch or Illustrator into a structured graphical environment like Charticulator (https://charticulator.com) or Drawing Dynamic Visualizations (https://vimeo.com/66085662).
