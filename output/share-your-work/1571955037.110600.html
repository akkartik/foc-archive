# Edward de Jong 2019-10-24 15:10
Just posted a handy color picker utility web app written in beads that lets you copy a color name or Hex value to the clipboard. It sorts the colors nicely, and resizes to a very wide range of window sizes. I believe that it is within 20% of the theoretical minimum program word count to achieve this task. 0 external dependencies, and only about a dozen functions total, how much simpler can it get? Anyway please check out the code at https://github.com/magicmouse/beads-examples/tree/master/Exa… or just use the utility at http://beadslang.com/apps/color_picker/picker.html

# Edward de Jong 2019-10-24 16:03
I used the fabulous TekTronix 4014 storage terminal screen back in 1977. It had an incredible 4000 x 3000 pixels, easily 10x what other devices had at the time. To erase the screen you had to press a zap button. The image was rock solid though, as it was etched in slow phosphors, so no flicker at all... It came with two clumsy thumbwheels for XY input, before they had trackballs and mice. Even more astonishing, it had a thermal printer that printed the screen for you in grayscale in a few seconds. Developed for the military, it was such great gear, and Tektronix was and probably still is a great company with excellent products.
