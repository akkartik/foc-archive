> This is known as the Yoneda lemma for preorders. The if and only if condition proved in part 3 implies that, up to equivalence, to know an element is the same as knowing its upper set—that is, knowing its web of relationships with the other elements of the preorder.

It's all about formalizing the "web of relationships." The ↑ summarizes the relationship of an element to the others, and ⊆ gives you the web so that you can recover the element from the summaries.

> The general Yoneda lemma is a powerful tool in category theory, and a fascinating philosophical idea besides.

It's a bit like the Database Reconstruction Theorem <https://www.youtube.com/watch?v=pT19VwBAqKA>: if you know all the statistics, you can recover the underlying data.
