# Andrew Reece 2019-01-26 03:56
I personally prefer analogue methods for productivity stuff:
- I have a 7x7 grid per habit (assuming I want to do the habit every day, Nx7 otherwise)
- I cross (X) off the day's cell when I've done the required thing.
- For habits with 2 components, I'll use / and \ on each cell (e.g. waking up by _: /, going to bed by _: \).
- I keep the bar for success low (e.g. read one page of a physical book) so that there's no barrier to starting - I find that once I've started I normally keep going.

(I think I first came across this technique on https://www.reddit.com/r/theXeffect/)
