# Mac 100ideas Cowell 2021-06-03 12:42
Hi all, I'm a biologist-turned-developer working on code experiments exploring how to help users create and reuse process documentation as they create narratives, notes, and files. Broadly I am interested in understanding how to apply the best practices and automation we take for granted when writing code in IDEs to the way people write semi-structured instructions intended to be "executed" not by computers but by other humans. I'm "100ideas" on most online platforms. See you around!
