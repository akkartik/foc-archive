# Kartik Agaram 2019-05-12 20:40
> There is no question that better languages can drastically simplify programming..

I question it, actually.

1. I don't think programming can be simplified. There's a kernel of irreducible essential complexity here. What we can do is unpack the complexity and stage it, so that newcomers don't have too steep a wall to climb. Carve steps into the sides of the wall.

2. I don't think the solution is better languages. The solution is better environments. We've recently discussed this here.

Totally fine that we disagree about these things! I just object to the "no question".
