# Emmanuel Oga 2020-11-25 18:54
I'm thinking for something like Tax rules having some redundancy because of lack of composability maybe is not a great deal, perhaps it is even a feature. Instead of saying "Tax rule #77 is like Tax rule #23 but with XYZ", you may need to copy and modify the old rule ... there would be no risk one rule changing modifying the other.
---
in any case for me the takeaway is that lack of composability can be alleviated with great tooling if the DSL provides enough advantages.
