# Beni Cherniavsky-Paskin 2025-05-21 10:49
If you find Scheme a barrier, and/or want to use your web background, there is also a JS edition nowdays: https://github.com/source-academy/sicp.
It uses small subsets of JS, and ends up implementing them, in same JS subset. And it lets you run code fragments interactively right inside the online book.
They also have a "comparison" edition where you'll see both original Scheme and new JS, and any diffs in explanations + some commentary on the conversion. That might be overwhelming, or insightful, or both :wink:
