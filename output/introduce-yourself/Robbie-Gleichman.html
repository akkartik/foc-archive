# Robbie Gleichman 2019-10-10 16:34
Hi, I'm interested in visual programming languages, and how they could make harnessing computers more enjoyable and accessible for everyone. I have been experimenting with represeting Haskell visually as a way to figure out what the biggest barriers are towards making visual programming languages universally useful: https://github.com/rgleichman/glance.
# Robbie Gleichman 2019-10-10 23:07
@Dan Cook Yes, the eventual goal is for Glance to become a code editor/debugger for Haskell or other languages. Currently, Glance drawings don't scale well to large and/or complicated functions, so I have been thinking about how to improve that recently. I think adding a more structured and linearized node layout (in addition to the current layout) with lines/wires routed in a more organized fashion might work. Right now I'm attempting to prototype the new layout by taking textual Haskell code and adding ASCII art lines to it.
