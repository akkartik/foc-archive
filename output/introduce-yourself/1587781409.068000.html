# William Taysom 2020-04-25 18:46
Hello Ian, yes text-free programming is an interest of mine not that I find any time to explore these days what with a newborn and the general state of affairs. I did write up a few thoughts a few months back https://gist.github.com/wtaysom/7e5fda6d65807073c3fa6b92b1e25a32. The main challenge with Kinetic programming as I call it is having a tractable mechanism for stepping back from the immediate performance in order to parameterize it. Well, perhaps I'm getting ahead of myself. Start with the medium of VR. What, ultimately, do you have to build with? The performer's motion, a mapping of time to position. Reify it. Transform it into light and sound. Those are the available feedback channels. An object, say a ball, is a derived concept: persistent geometry, translational symmetry, dynamics to be grabbed, to bounce.
