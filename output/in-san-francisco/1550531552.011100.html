# Edward de Jong 2019-02-20 21:23
Tyretool is basically a web version of Filemaker Pro without any of the graphical customization possibilities at $10/user/month, so slightly cheaper than filemaker pro pricing for small user bases, but a great deal more expensive for large numbers of users. Filemaker Pro may still have a poor scripting language, but it is extremely powerful, and i have used Filemaker Pro many times for small databases because it is so easy to work with, and so well documented at this point, being 20+ years old. Filemaker Pro is one of those products like Delphi that gets zero love because it is so simple.
