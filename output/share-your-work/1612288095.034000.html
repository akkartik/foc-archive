# Florian Cäsar 2021-02-02 09:48
I made an experimental interactive & adaptive history of natural language processing (the data science subfield) - but the point isn't the topic, the point is how you can interact with the content, which is an example of what I want to do with all kinds of content - you can view and query it from any perspective:
• get a general summary
• see how a specific technique developed over time,
• explore varying levels of detail,
• jump around associatively,
You can also just start from scratch and make completely new queries (and share a link here for the rest of us).
I'm very curious what you all think of interacting with content like that. Imagine if you could interact with all kinds of content - web content, standard documents, code docs - with an interface like that.

(I hope this is relevant, if it isn't, I'll happily delete it :))
