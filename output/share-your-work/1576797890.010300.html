# Will Crichton 2019-12-20 11:14
Thanks @Maeliza. The primary set of commands is what’s listed. I think about it as a programmable-compiler---it takes standard optimizing compiler techniques (function inlining, copy propagation) and exposes them to the user in a human-readable way. Currently, my goal with this tool is to make APIs more accessible by enabling programmers to see what’s happening without reading through all the source code.
