# Luifer De Pombo 2020-11-16 08:10
I created a new programming language with a friend to see if we could create a compiled language with a simplified bytecode, wrt to traditional languages which allow arbitrary computation that is not guaranteed to halt, that could be automatically parallelized by a VM/runtime (written in Rust :crab:) based on the size and shape of the data. It is still very much in an alpha state, but lmk if you want to learn more or have feedback for us!
• A blog post we wrote about what Turing Completeness prevents https://alan-lang.org/the-turing-completeness-problem.html and HN discussion https://news.ycombinator.com/item?id=25006619
