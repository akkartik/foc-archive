# William Taysom 2019-09-25 17:48
>> 2. Show that this construction defines a monotone map ↑: op(P) → U(P).

More definition unwrapping. U(P) is the set of upper sets in P ordered by set inclusion ⊆. For ↑ to be a monotone map, ↑ needs to firstly take every element of P to an upper set in P – we showed that in part (1) – and secondly if q ≥ p then ↑(q) ⊆↑(p). Let's prove it.

Take q ≥ p. This means p ≤ q. Consider r ∈ ↑(q). This means q ≤ r. By transitivity p ≤ r. This means r ∈ ↑(p). And so ↑(q) ⊆↑(p).

The logic rhymes with part (1).
