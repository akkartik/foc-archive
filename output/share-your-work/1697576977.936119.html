# Mike Austin 2023-11-18 16:09
Thank you, @Yoz Grahame! React Desktop and Kopi are separate projects - I just recently started integrating Kopi into it, exploring different ways to use Kopi. Both are written in TypeScript. It's a fun project since there are no goals per se, just exploration. So far you can use the Terminal as a Kopi REPL/CLI, read and modify the examples in Learning Kopi.md, and run the examples in the Scripts folder (Particle System.kopi, etc.). There's also "Designer", a very early prototype to a "Flash" type editor. I'm trying to think very simple UX to create simple apps and games.
