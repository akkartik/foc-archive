# Paul Tarvydas 2026-04-14 07:25
Some thoughts about The Computing Historian Recovering Ideas We Threw Away

• we finally have asynchronous multi-tasking - i.e. the internet - many independent computers, running at their own speed
• we have some of the same “concurrency” issues, with internet, yet simpler, since internet does not allow actual memory sharing (you can share data, but not memory)
• modern computing has been overrun by technical/academic complication, not by UX simplicity
• the “new” invention of CompSci is not “programming” but “re-programming”
• Design Recovery to lift designs out of old software and to re-express them in code for new RMs - what is the meta-language? (factbases of triples?)
