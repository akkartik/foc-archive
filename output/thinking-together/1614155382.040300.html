# Jack Rusher 2021-02-24 01:29
It's a deep topic that I don't have time to type about very much atm, but the two lines of inquiry that have worked best for me have been:
• Training systems to align labels with other things in a shared latent space. In an art context, that can show itself as a model that can caption images also being able to direct a GAN to generate images based on a caption, as I did here: https://twitter.com/jackrusher/status/1356902381880025089
• Models that generate computer programs representing their decision process, which gives more of the kind of procedural representation that is sometimes required. (No link because I haven't published any of this stuff).
In the "AI" + FoC space, I'm probably most excited by things like this, which can lead to more human-friendly coding assistants.

# Garth Goldwater 2021-02-24 14:25
forth has the delicious property of being trivially serializable into tokens because you can read each token as a function that takes a stack and returns a stack, and space as both the token separator and the function composition operator. so you can split a forth program anywhere and replace n tokens with a word in a lookup table that returns the original string of tokens and get the same result (i’m sure there are edge cases, but it’s pretty universal)
