Also interesting, was wondering what "higher order unification" was. Nice example:

f 3 == 3 + 3

Higher order unification: 4 solutions

\ x -> x + x
\ x -> x + 3
\ x -> 3 + x
\ x -> 3 + 3

In contrast, first-order unification/matching would give no solution.
