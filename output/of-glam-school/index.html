# Tudor Girba 2020-10-11 12:41 #
So, to start getting familiar, I suggest the following:
• Download GT: gtoolkit.com/download
• Go through the tour slideshow from inside the environment (follow the yellow sign)
• If you are new to Pharo, please follow the Pharo 101 slideshow, too
But, more important would be to pick a specific problem to play with. Some areas that can be interesting:
• take a business domain to model and visualize through custom inspector views
• explore a remote REST API from the inspector
• try to assess a problem in a system already supported like JS/React or Java
• build support for a language (best to start with a small one)
• explore the construction of a visualization combined with live widgets
