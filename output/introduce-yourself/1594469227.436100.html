# Edward de Jong 2020-07-11 13:34
Welcome to the group. You will find that the main reason mobile apps are so large is that they rely on too many framework and modules, which because of their interconnected/tangled dependency chains result in gigantic code bases (in total). But actually the mobile apps are quite small in their core function. You will find many projects on the spreadsheet for this group that are tackling the central problems from many different angles.

All programming is abstract algebra, which is a fairly new field as far as mathematics goes.
