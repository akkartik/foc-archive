# Tyler Adams 2020-05-17 03:55
@Chet Corcos Config (with defaults) is used to enable a tool to solve a bounded problem easily. Composition, which may include boiler plate, is used to enable a collection of tools to solve unbounded problems. A unix pipeline is a good example of managing the tension properly. Each step in the pipeline is a command with configurable options that easily solves a bounded problem (ex grep finds lines matching a regex), and unix itself permits composition to solve an unbounded problem (a pipeline runs an arbitrary program).
