The Proof is in the Universal Frontend

Tutuca ships with everything required to instantiate any application "Just In Time" in the browser without any global state:

- Components can compose without knowing about each other using a "Component Selector"
- Compile and register components on the fly
- Compile Tailwind classes at runtime
- Register custom CSS & Web Components
- Listen to new and custom events
- Manage lifecycle events
