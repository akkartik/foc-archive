# Kartik Agaram 2022-07-30 09:00
Scrolling file at a time on the infinite 2D surface of my note-taking app.

The problem I was trying to solve was that I'd scroll down some ways, open a new file and find the new, short column way off screen above. Now I can bring the new file on screen with with one more, top command. I'm still not quite happy with this solution. I don't want to always automatically jump back up to the top of the surface; that feels more busy than the feeling of solidity/reliability I'm going for.
