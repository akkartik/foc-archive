# Laura 2018-12-12 09:11
The colors are indeed types of AST nodes, I've put 0 thought into how they are colored, the color is simply

"#" + intToRGB(hashCode(this.node.type))
# Laura 2018-12-12 09:13
I'm also not using an intelligent layout, edges overlap because I'm using a homebrew, very, very simple recursive rendering algorithm. It's what you would use to render a binary tree, but a varying amount of child-branches instead of 2.
The only thing I added was that if there are more than 10 branches, then I wrap all of them around in a circle, I chose this randomly, it worked fairly well to pin down hotspots
# Laura 2018-12-12 09:13
The visualisation itself is very rudimentary and nothing more than a proof-of-concept
# Laura 2018-12-12 09:17

to allow for indefinite nesting of hierarchical structures
# Laura 2018-12-12 09:18
