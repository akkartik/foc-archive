I have seen a computer game that lets you program in an isometric-view 3D world, with logic gates, etc., super fancy and clever, so it is possible.

Working on small screens is quite cumbersome, and debugging is just infeasible. Debugging a graphical interactive product on a single monitor even 32" in size is a tricky thing (I do it all the time).

So just because the kids are used to working on phones doesn't mean it is practical or sensible. It requires way too much human memory to do it that way.
