# Sébastien 2019-05-16 05:59
Hi there! I'm Sébastien and am currently based in Montréal (moving to Wellington, NZ in a few months). I've been interested in PL design since late 90s and have created a few transpilers ([Sugar](https://github.com/sebastien/sugar), [Sugar 2](https://github.com/sebastien/sugar2), [PAML](https://github.com/sebastien/paml), [PCSS](https://github.com/sebastien/pcss] as well as underlying tools/libraries like [LambdaFactory](https://github.com/sebastien/lambdafactory) and [Libparsing](https://github.com/sebastien/libparsing)). My background is primarily in software engineering, but I also have studied design and have worked at the intersection of both creating interactive data-visualizations and data-rich interfaces at [FFunction](https://ffctn.com)

I'm currently working on two PL-related projects at the moment: Curved, a visual editor for the Curv Language by Doug Moen (who's also on this forum) and TLang, a language for incremental tree processing to build compilers, build systems, reactive user interfaces, databases and other applications that rely heavily on a graph structure.

I'm very interested in the topics of visual programming and how programming languages/paradigms shape our mental model, and I'm particularly interested in the technical topics of parsing, abstract program model/program representation and data structure synthesis (all of which, you can guess, are quite useful in building modern compilers!).
