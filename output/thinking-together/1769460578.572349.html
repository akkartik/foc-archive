# Federico Pereiro 2026-01-26 12:49
I wrote a short article prompted by Ryan Dahl’s statement that “the era of humans writing code is over.” (I consider Ryan a living legend, BTW).

I have two objections/doubts:
1. What if code is no longer writing “syntax”? What if code can be as readable as natural language?
2. How are we supposed to review code if we don’t write it? Can you be an editor without being a writer? Can we consider LLMs as good enough code writers that take natural language descriptions of a system, in the same way we consider compilers “good enough” machine code generators?
I’d like to hear what you think (or feel) about this.
