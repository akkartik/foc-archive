# avon 2023-10-23 13:07 #
I also want to take the chance to post some FoC-related communities in the Philadelphia area:
• Iffy Books: hacking, free culture, gardening, zines. So thankful for this community, their monthly Permacomputing Meetup is a great event too.
• Philly Community Wireless: Building community-controlled net…. Community-run infrastructure opens computing possibilities which are not possible under more traditional ISPs. The team behind PCW is also wonderful :heart:
• Philly JS Club: a place for developers of all skill and expe… Haven't personally been to a hack night yet but it seems like a good time.
This is just a start, please reply in thread with any additional communities! It would be cool to have a living archive of Philly ~FoC communities in this channel.
