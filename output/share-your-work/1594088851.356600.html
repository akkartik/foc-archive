# Maeliza 2020-07-06 19:27
Hey folks,
With @Shubhadeep Roychowdhury we just released the last version of our Future of Coding projet codeBert a package to automatically review Python code documentation (docstring)
We'd love to have your feedback on it: codeBERT github package

This is all part of a longer-term ambition: automatically document any source code. This is our plan:
:heavy_check_mark: automate source code mining   with tree-hugger
:heavy_check_mark:  masked language model trained on source code (Python) with codeBERT MLM
:arrow_forward:  fined tuned language model to analyse docstring vs function definition (Python) - and improve it
:black_circle_for_record:  extend model to different languages
