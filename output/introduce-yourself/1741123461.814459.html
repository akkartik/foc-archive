# Guyren Howe 2025-03-12 16:28
No amount of lipstick on the pig that is SQL will make it any prettier. Folks have spent a great deal of effort making all kinds of SQL clients but folks don’t use them much because they’d rather do data manipulation in anything else if they have any choice.

This is the biggest single problem with the computer industry. We ought to be leaping to do our data manipulation using the Relational Model — that’s what it’s for. It’s the ideal paradigm for doing it.

But you have to use SQL, which is so bad, it’s punishment for shoplifting in some countries.
