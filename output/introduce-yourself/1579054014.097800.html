# westoncb 2020-01-14 19:11
I did a lot of experimentation with that in-between level before arriving at the jump to lines—which I really didn't want to use since I was having to break the more elegant mapping to grammatical units. My memory is that it wasn't often desirable to navigate the detailed hierarchies that would show up there.

Code blocks sounds reasonable though—I certainly had them included at one point :slightly_smiling_face: e.g. dragging a for loop around or whatever.

My ending feeling was still that I hadn't found an optimal breakdown of units that should be made Tiles in the UI. I hadn't seen any other editors like this at the time, so I sort of blindly assumed that directly mapping the "Tiles" to the grammar would be an optimal (or at least non-problematic) UI.
