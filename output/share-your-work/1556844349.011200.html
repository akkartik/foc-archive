# Corey Montella 2019-05-02 17:45
Hi everyone. I've made some progress on Mech documentation I wanted to share here. Mostly I've done work on the rendering of Mech documents, so I've just started with writing the actual documentation. It's rough and incomplete now, but I was hoping for some feedback on the docs layout.

Here's a good example of a page that shows off most of the features:
http://docs.mech-lang.org/#/docs/programs/blocks.mec
Here is the raw text:
https://gitlab.com/mech-lang/docs-page/raw/master/public/doc…

Couple of features:
-You can render a view under a block by writing to the block/view table. The syntax is subject to -change.
-You can embed a Mech value in text with double square brackets. [[]]
-You can click on any table in a block to see a table inspector with the contents rendered in a nice layout

Here's an example of a function reference:
http://docs.mech-lang.org/#/docs/math/sin.mec

Here's an example of an error:
http://docs.mech-lang.org/#/examples/error.mec

Hoping for any feedback you might have before I go back to writing the actual documentation. Thanks!
