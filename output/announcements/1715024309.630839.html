# Ivan Reese 2024-05-06 12:38
FoC Virtual Meetup
Thursday May 23rd • 16:00 UTC

@Maikel van de Lisdonk has offered to host a virtual meetup — the first in a good long while, but hopefully something we'll start doing more frequently. This meetup is a remote-friendly way for members of our community to get together, talk about our common interests and ideas, and share a few demos. If you'd like to attend, please RSVP by clicking the :elephant: reaction on this post. All are welcome!

If you'd like to attend and you'd like to give a short demo (15 mins max including Q&A) of something you're working on, please add a reply in the thread for this post.

It's very likely that the meetup will be recorded and then posted on YouTube, so if you're unable to make it but would like to catch up on what was shared, that option should be there.

Looking forward to seeing some familiar faces and hopefully a bunch of new ones too!
