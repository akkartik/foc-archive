# Tibor Halter 2019-10-25 10:00
Oh, btw, I built what I call a Posix Bridge for Boomla. It translates between the Boomla and the Posix filesystem, making it possible for you to use your own tools. There is an SFTP server built into the binary distro that lets one do that. Also, there was FUSE support so that you could mount a Boomla website into a directory. But that was buggy and nobody used it, so I removed it. There is also an import/export capability.
