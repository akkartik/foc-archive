# Paul Tarvydas 2025-07-04 12:15
Very interesting. Thanks! I watched the video in the Welcome window. First impressions:
1. I wish it were 3D. Layers.
2. Multiple outputs would be nice. Output could be JSON, e.g. console.log("Hello"); produces {"stdout": "Hello"}. This would allow "fan-out" - feed the same output to multiple receivers, letting each receiver pick and choose outputs that it wants to deal with. Our function-based mindset implies that there is but one output, hence, output doesn't need to be tagged, something which I find to be very restrictive.
