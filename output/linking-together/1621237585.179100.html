# Jack Rusher 2021-05-17 00:46
The 2021 edition of HotOS (hot topics in operating systems) program was just wrapped up (conference is in a week or so). This paper on Aurora, an experimental single level store, seemed like it would be interesting for this audience:
> We present the Aurora single level store (SLS), an OS that simplifies persistence by automatically persisting all traditionally ephemeral application state. With recent storage hardware like NVMe SSDs and NVDIMMs, Aurora is able to continuously checkpoint entire applications with millisecond granularity.
