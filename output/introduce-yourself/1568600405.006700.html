# Luke Persola 2019-09-18 21:49
There’s a one-to-one relationship between the colored blobs (“balloons”) and nodes in the AST. I’m not sure how obvious the tree structure is to fresh eyes but the child nodes’ balloons are supposed to look like they’re inside their parents’. The balloon color represents syntactic structure and the text color is supposed to be type, but I haven’t gotten there yet. At any time you’re focused on one node in the structure, which conditions which editing commands are available. Right now I’m trying to peel apart the editor from the test language I built it alongside.
