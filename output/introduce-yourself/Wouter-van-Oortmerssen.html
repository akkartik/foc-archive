# Wouter van Oortmerssen 2019-03-27 16:33
Hello, World! I'm Wouter van Oortmerssen and I've made a few programming languages (http://strlen.com/programming-languages/) probably the most futuristic being Aardappel (graphical distributed tree rewriting by example, http://strlen.com/aardappel-language/) and Restructor (fully automatic whole program refactoring, also graphical, http://strlen.com/restructor/). My main project right now is a more conventional language, Lobster, that nevertheless has some innovation (e.g. flow sensitive type specialization, automatic lifetime analysis, FRP-like frame variables etc, http://strlen.com/lobster/). On the sillier side I once caused an esolang revival with http://strlen.com/false-language/. Oh and in my day job I work on LLVM and WebAssembly :slightly_smiling_face: Also make games and game engines, and productivity tools like TreeSheets. Details: http://strlen.com/
