# Mo 2020-10-29 06:29
Yes, yes and yes. If you're up for it, shoot me a DM and we'll find a time to chat and demo. I need feedback so happy to share all my work as I go. Answers below:
• Closer to Parse Cloud (before FB shut it down), but right now, primarily for web apps.
• Visual database designer that gives you both structured and unstructured (JsonB) fields in PostGres, gives you CRUD and custom API endpoints in both REST and GRAPHQL
• Ability to build custom endpoints without writing tons of code
• Admin view builder (not great for consumer facing views, but great for b2b apps, though anyone can use the APIs to build their own frontend if they like).
