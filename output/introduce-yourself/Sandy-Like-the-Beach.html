# Sandy Like the Beach 2026-03-11 10:57
Howdy; long time listener, finally decided to get in touch with the community. I've been working on a combinator library for constructively composing music, in the intellectual stylings of Conal Elliott, Dmitri Tymoszko and Robert Gjerdengen. It's currently a DSL sort of thing (because I'm a PL guy), but more and more thinking that a nicer UI would go a long way. I have some rough ideas on what that might look like, but haven't found any easy way to prototype (off the shelf tools don't seem to work very well for 4+ dimensional spaces).

Anyway, assuming the community is anything like the podcast, I'm really excited to meet up with some like-minded folk.
