# Oleksii Shmalko 2020-11-09 11:53
I write an imgui library for my editor (https://github.com/rasendubi/meta/tree/master/meta-rs/meta-gui). A good source of inspiration was Go's Gio library. One idea I've stolen is that drawing operations are not immediately applied but are stored first, so you can apply transformations after the fact. In particular, this allows implementing automatic layout with constraints. Gio has a good overview at https://gioui.org/doc/architecture
