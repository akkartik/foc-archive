# nicolas decoster 2020-02-09 13:14
I am also interested by offline first experience. In my case, for programming environments. I played a bit with offline first text editing using CRDT or, more specifically, using CausalTree. I have worked on a proof of concept based on Rust (for the CausalTree part) compiled into WebAssembly and used into a very raw text editor in Vue.js. My purpose was to give a talk on WebAssembly to the local Rust community. So, it is no more than an experiment but it was very fun to do and it makes me understand some aspects of CRDT.
