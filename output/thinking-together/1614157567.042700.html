# Cole Lawrence 2021-02-24 02:36
This felt like the best application of Clojure I've ever experienced. The dynamic nature of the language made it easier to rollout new versions without worrying about state machine type incompatibilities between versions.
Then, Clojure's meta programming capabilities were awesome for incrementally improving our "DSL" for specifying chatbots.
