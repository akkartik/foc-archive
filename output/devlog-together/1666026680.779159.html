# Jason Morris 2022-10-17 10:11
Ever have one of those days where you realize the thing you are making is very possibly actually a completely different thing? Like, Blawx will allow you to run a query, in the absence of concrete facts, and will find all the various different reasoning paths that could be used to answer that query, and show them to you. If you recombine those explanations using disjunctions, you would have a single decision tree. Invert it, that tree is an imperative function. So... Blawx is maybe a tool for generating imperative code from declarative specifications, where the declarative specification is a law? :exploding_head:
