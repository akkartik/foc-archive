# t400 2023-07-17 08:05
Hi all I am Avon, my background is in music (guitar + pure data) and delay tolerant networking. I actually found future of coding through the podcast episode with pd creator Miller Puckette.
My current interest is in building collaborative, itemized, computing environments, inspired by the writings of Alexander Obenauer. The current path I'm taking towards this goal is working with Project Mage, and trying to connect Mage's data representation system called KR, to a delay-tolerant data synchronization protocol called Bramble. You can read more about this idea here: https://lists.sr.ht/~project-mage/discussions/%3C6fb9dbc5-1c…

My overarching goal with this work is to build the tools necessary to create multi-computer, collaborative computing environments. Initially this is just going to be software, but eventually I would like to branch into physical environments and explore how this software interacts with alternative human computer interfaces like Dynamicland, possibly paired with modular building systems like OpenStructures and GridBeams.

Feeling really grateful that I found a community of others interested in similar ideas. :smiley:

Cheers,

Avon (@t400@iffy.info)
