# Yousef El-Dardiry 2021-02-03 12:10
Thanks!! Was a little nervous to share a first video ;)

Observable is definitely a big inspiration. I started with the project to see what observable with first-class typescript+react support could look like.

I’m also doing some experiments to go beyond the traditional “notebook” use cases. Hope to share more on that later.


For now, the main differences are:
• evaluate as you type
• Built on top of Monaco (VS Code).
• Import any module from npm (including types), uses Skypack
• First-class typescript / javascript support (observable introduces some new keywords, id like to prevent that)
• React / tsx support
• change language of cells (now markdown + typescript are supported)
• The “reactive part” is built on MobX
