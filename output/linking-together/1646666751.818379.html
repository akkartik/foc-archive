# Orion Reed 2022-03-07 08:12
I love this aesthetic, I often write small TUI programs and have always used curses, ncurses, or their descendants. This looks like it might be a good replacement! There’s something very compelling about a character grid. I think of it like a low-res pixel array but each pixel can also be a symbol/glyph.

Wouldn’t have the same feel at all if it wasn’t monospace because you lose the grid, though changing the aspect ratio of the “pixels” seems fine, which seems to be around 1:2 on average.
