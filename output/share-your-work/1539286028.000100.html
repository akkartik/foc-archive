# Corey Montella 2018-10-12 12:35
My very simple advice is to make sure it's a good tool. We had trouble with that in the Eve that used NL queries. People didn't know how to use it because it was too fuzzy. Tools need to be predictable and have a very straight forward application. Natural language seemed like a panacea but was actually a can of worms, because it's soooo open ended to the point it's paralyzing. I always thought that if we directed the users with some ambient tooling that helped them refine a query as they were typing it would go a long way toward making NL a better tool.
