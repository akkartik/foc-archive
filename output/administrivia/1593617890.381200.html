# Konrad Hinsen 2020-07-04 02:41
@Joe Nash Your comment is both right and wrong, on different levels. Yes, the comment implies what you say. But why is this a problem? Because it creates negative feelings in some readers.
If the implication were obviously wrong, it would be used jokingly. If it were obviously right, we wouldn't been discussing it. "Obviouly" in these statements means "in the eyes of pretty much the whole population". The problem is that it used to be obviously true in the past, as part of a world view that is increasingly criticized for the conclusions it supported. More and more people thus want it to be wrong, or at least irrelevant, and bringing the claim up thus creates negative feelings. But there isn't much of a rational basis for proving the claim wrong (there are more science-based arguments for it being true to some degree), so the option we are left with is declaring it taboo. Which is in fact a workable solution in real life: if you make this claim in a group that considers it inapproriate, there's an awkward silence and everybody stares at you - you will then think twice about doing it again. But we haven't found good techniques for sanctioning taboo violations in on-line discussions so far.
