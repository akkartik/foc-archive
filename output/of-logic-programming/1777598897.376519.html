# Jason Morris 2026-04-30 18:28
In December I released Blawx as SaaS. It's a block based logic programming environment for encoding legislation. A couple of weeks ago I released a local MCP server for it so you can get your coding agent or Claude Desktop to do the visual encodings for you. By this weekend I should have a remote MCP equivalent live, so anyone with a paid Blawx account and a chat agent that supports remote MCP will be able to use agentic AI to do advanced (sCASP) logic programming encodings of legislation. I have about 100 things I want to try after that, including using the logic encodings to generate test harnesses to build compliant software in other languages, better validation workflows, etc. But I have limited time to spend on it. So we haven't ALL forgotten about logic programming.
