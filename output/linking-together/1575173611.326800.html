# Doug Moen 2019-12-02 12:30
Reimplement this idea using special lego blocks. The lego blocks contain electronic circuitry, so they actually implement the logic. To support copy/paste, implement reflection: There is a block that operates like the "quote" operator in Lisp: It constructs a value describing the construction that the Quote block is plugged in to. Then attach an Email block to the Quote block and email the program to yourself.
